## Git?

你可能经常遇见带Git的工具或者应用，比如Github,Gitbook,Gitee。

Git是一个分布式版本控制工具。由Linus开发，用于维护Linux版本。

有了Git,你可以控制你的代码版本，比如某个版本的代码出现了致命错误，你可以回到以前的版本。

这里不过多介绍，Git太过于常见了，甚至交作业，做科研都会用到。That's it.
